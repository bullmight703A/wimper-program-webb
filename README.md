# W.I.M.P.E.R. Program - WordPress Theme

This is the dedicated repository for the W.I.M.P.E.R. Program WordPress theme. It is a standalone Single Page Application (SPA) designed for financial architecture and payroll tax optimization.

## Key Features
- **Section 125/105 Chassis**: Built-in logic for FICA tax reduction modeling.
- **SPA Architecture**: Smooth navigation without page reloads.
- **High-Performance**: Lean codebase with optimized assets and lazy-loading.
- **Brand Identity**: Custom navy and gold secondary palette with "mathematical" contrast.

## Deployment
This repository is connected to the production server via WP Pusher. Pushing to the `main` branch will trigger an update on the live site.

## Development
- **Styles**: Built with Tailwind CSS and Font Awesome.
- **Scripts**: Vanilla JavaScript for hydration and interactivity.
- **Framework**: WordPress 6.x compatible.
